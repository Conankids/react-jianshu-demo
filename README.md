
## 项目指令

### `yarn start`

进入开发环境，打开浏览器[http://localhost:3000](http://localhost:3000)查看项目

### `yarn build`

打包生产环境资源

### `yarn eject`

释放webpack配置文件，（一次性操作，操作后不能复原项目）
